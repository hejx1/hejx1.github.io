概述

位运算是以二进制位为单位进行的运算，其操作数和运算结果都是整型值。
位与’&’，位或’|’，位非’~’，位异或’^’，右移’>>’，左移’<<’，0填充的右移’>>>’
位运算的位与’&’，位或’|’，位非’~’，位异或’^’与逻辑运算的相应操作的真值表完全相同，其差别只是位运算操作的操作数和运算结果都是二进制整数，而逻辑运算相应操作的操作数和运算结果都是逻辑值boolean型。
